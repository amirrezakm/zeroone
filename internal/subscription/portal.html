How to import

📱 Hiddify Next (Android / iOS / Windows / macOS)
Add to Hiddify
  1. Open Hiddify Next.
  2. Tap + → Add From URL.
  3. Paste the subscription URL above and confirm.
  4. Pick a server from the list and connect.
🖥 v2rayN (Windows)
  1. Open v2rayN → Subscription → Subscription group setting.
  2. Click Add, paste the subscription URL into URL, give it a remark.
  3. OK → then Subscription → Update subscription content (Without proxy).
  4. Double-click any server in the list to activate it.
📦 NekoBox / NekoRay (Windows / Linux / Android)
Add to NekoBox
  1. Open the app → Subscription Groups.
  2. New → set a name and paste the subscription URL.
  3. Save, then right-click the group → Update.
📱 V2RayNG (Android)
  1. Tap the menu (top-left) → Subscription group setting.
  2. Tap +, paste the URL, give it a name.
  3. Back → menu → Update subscriptions.
  4. Pick a server and connect.
🍎 Streisand (iOS)
Add to Streisand
  1. Tap Add to Streisand above; the app opens with the subscription pre-filled.
  2. If it didn't open, copy the URL and paste it via Add Profile → Subscribe.
  3. After it refreshes, pick a server and connect.
🍎 Shadowrocket (iOS)
Add to Shadowrocket
  1. Tap Add to Shadowrocket above; the subscription is imported automatically.
  2. Open the app → Subscribe tab to see the imported profile.
  3. Pick a server and connect.
🍎 FoXray (iOS)
  1. Copy the URL using the button above.
  2. Open FoXray → + → Subscribe → paste the URL.
  3. Tap Save, then pull-to-refresh to update servers.
📱 Karing (iOS / Android / Windows / macOS)
Add to Karing
  1. Tap Add to Karing above; the app opens and imports the subscription.
  2. If it didn't open, copy the URL and paste it via Profiles → Add → From URL.
  3. Pick a server and connect.
🧪 Clash Meta / Mihomo / Clash Verge
Add to Clash
  1. Tap Add to Clash above to install the profile in one tap.
  2. Or open the app → Profiles → New Profile from URL and paste the URL.
  3. Select the ZeroOne group and choose a server inside it.
If your subscription link is exposed and needs rotating, contact your admin.
